# Contributing

Thanks for helping build AltmanAI's MasterPlan.

## Ground rules
- Keep it human-first: dignity and well-being first.
- Keep it verifiable: prefer receipts, hashes, and clear references.
- Keep it small: ship increments.

## Ways to contribute
- Fix docs or clarify language
- Add tooling improvements under `tools/`
- Add public-safe impact records under `impact/records/`
- Propose issues with tight acceptance criteria

## Pull request checklist
- [ ] I understand and follow `CODE_OF_CONDUCT.md`
- [ ] My change is public-safe (no secrets, no private personal data)
- [ ] I updated docs where needed
- [ ] I ran the relevant tools or checks (see below)

## Local checks
```bash
python3 tools/validate_repo.py
python3 tools/run_self_test.py
```
