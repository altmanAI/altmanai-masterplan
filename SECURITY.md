# Security Policy

## Reporting a vulnerability
Please do **not** open a public issue for security bugs.

Email: security@altmanai.tech  
Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix

## Coordinated disclosure
We aim to acknowledge within 72 hours and work toward a fix as quickly as reasonably possible.

## Scope
This repo includes docs and small tooling scripts. If your report affects another AltmanAI repo, include links and details.
